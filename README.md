# COGIMOT

Jeu de devinettes de mots français, jouable sur smartphone et ordinateur.

## Fonctionnalités

- Deviner un mot français en 6 tentatives (ou plus selon les réglages)
- 4 à 8 lettres au choix
- Mode infini pour les enfants
- Retour visuel : vert (bien placé), orange (mal placé), rouge (absent)
- Clavier AZERTY optimisé tactile
- Auto-soumission à la dernière lettre
- Mode sombre uniquement

## Jouer

Ouvre `index.html` dans un navigateur, ou déploie sur GitHub Pages.

```
https://toncompte.github.io/cogimot/
```
